Homebase

A control center for recurring desk workflows. Homebase turns daily setup, uploads, workbook actions, browser positioning, and routine desk operations into one repeatable system instead of a pile of memory-dependent steps.

Workflow coverage

The system automates more than fifteen daily workflows, including Excel report generation, macro execution, ERCOT schedule uploads, PHAROS and PJM processes, browser layout management, and morning desk setup on Windows.

  • Desktop UI built for live use.
  • COM-driven Excel automation where it actually matters.
  • Portal workflows and browser orchestration where external systems still drive the process.
